# PayTrack - Invoice Management System

A full-stack invoice management application that allows users to create, edit, view, delete, and track invoices with detailed client information, itemized billing, and status management (Pending, Paid, Draft).

## Features

- User authentication (Signup & Login)
- Create new invoices with custom invoice IDs
- Edit existing invoices
- Delete invoices (with full data cascade)
- Mark invoices as **Paid** or keep as **Pending**
- View all invoices with total amounts
- Detailed invoice view with:
  - Sender & Client addresses
  - Payment due date & terms
  - Itemized list with quantity, price, and totals
- Responsive design ready (backend fully supports frontend integration)

## Tech Stack

### Frontend 
- React / Vanilla JS 
- Tailwind CSS and Framer Motion

### Backend
- Node.js + Express.js
- MySQL database
- RESTful API architecture
